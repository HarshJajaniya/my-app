## 📦 Installation & Setup

Clone the repository:
```bash
git clone https://github.com/HarshJajaniya/my-app.git
cd my-app
````

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm start
```
